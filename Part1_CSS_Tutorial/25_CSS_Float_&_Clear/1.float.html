CSS Float Property Example

Float: Left

This image is floated to the **left** of the text. The text wraps around the right side of the image, filling the remaining space in the container.

Float: Right

This image is floated to the **right** of the text. The text wraps around the left side of the image, filling the remaining space in the container.

Float: None

This image has **no float** applied. It appears in the flow of the text, just as it occurs naturally in the document.

Float: Inherit

This image inherits the **float: left** property from its parent container. The text wraps around the image accordingly.
